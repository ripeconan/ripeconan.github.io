plot 函数的参数

plot 函数的若干参数: ?plot, ?plot.default

  • type 可以画点, 画线, etc.
  • main 标题
  • xlab, ylab 横纵轴标题
  • xlim, ylim 指定横纵轴的取值范围
  • col 颜色
  • pch点型, cex点大小, lty线型, lwd线宽, 等等

很多其他高级图形函数也有这些参数
